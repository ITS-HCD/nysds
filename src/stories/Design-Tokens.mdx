# Design Tokens

Excelsior components are built using a set of design tokens. In this design system, design tokens are variables in Figma and CSS that represent core design choices. These are used to build components that are consistent between design and code and across platforms.

<style>
  {`
    /* Static styles */
    table {
      border-collapse: collapse;
      width: 100%;
      height: 100%;
      /* No border */
      border: 0 !important;
    }
    td {
      background: #fff !important;
    }
    .swatch {
      display: block;
      border: solid 1px var(--nys-color-neutral-300);
      border-radius: var(--nys-radius-md);
      width: var(--nys-size-1000);
      height: var(--nys-size-600);
    }
  `}
  {tokens.map((token) => {
    if (token.type === "color") {
      return `
        .swatch-${token.name.replace("--", "")} {
          background-color: var(${token.name});
        }
      `;
    } else if (token.type === "space") {
      return `
        .swatch-${token.name.replace("--", "")} {
          border: 0;
          margin: 0 auto;
          border-radius: 0;
          border-top: dashed var(--nys-size-1px) var(--nys-color-theme-mid);
          border-bottom: dashed var(--nys-size-1px) var(--nys-color-theme-mid);
          border-left: solid var(--nys-size-2px) var(--nys-color-theme);
          border-right: solid var(--nys-size-2px) var(--nys-color-theme);
          width: var(${token.name});
        }
      `;
    } else if (token.type === "size") {
      return `
        .swatch-${token.name.replace("--", "")} {
          border: 0;
          border-radius: var(--nys-radius-round);
          margin: 0 auto;
          width: var(${token.name});
          height: var(${token.name});
          background-color: var(--nys-color-theme);
        }
      `;
    } else if (token.type === "radius") {
      return `
        .swatch-${token.name.replace("--", "")} {
          background: #f0f0f0;
          border-radius: var(${token.name});
          border: 1px solid #ccc;
        }
      `;
    }
    return "";
  }).join("")}
</style>
