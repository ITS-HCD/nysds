# Welcome to the Excelsior Design System

Welcome to **New York State's Excelsior Design System** Storybook. This is where you can explore, experiment with, and contribute to the **components**, **patterns**, and **guidelines** that make up the design system.

## What is the Excelsior Design System?

The **Excelsior Design System** is a set of **reusable web components** designed to create consistent user experiences across New York State's digital products.

## Key Features

- **Lit-based Web Components**: Built on the powerful and lightweight Lit framework.
- **Scalable Architecture**: Designed for long-term scalability and flexibility.
- **Vite for Modern Builds**: Leveraging Vite for fast development and production builds.

## How to Use

- Navigate through the components using the **sidebar**.
- Experiment with the **controls** to customize components in real time.
- Use the **documentation** tabs to read more about component usage and best practices.

## Getting Started

To start using the components, explore the available components in the sidebar. For each component, you'll find:

- **Live previews** of each component.
- **Interactive controls** for changing properties and styles.
- **Documentation** on best practices for using the components.

Let's build together to improve the digital experience across New York State!
