## Events

The `nys-tooltip` component emits the following events:

1. `click`: Emitted when the tooltip is clicked.
2. `focus`: Emitted when the tooltip receives focus.
3. `blur`: Emitted when the tooltip loses focus.

You can listen to these events using JavaScript:

```js
// Select the tooltip component
const tooltip = document.querySelector("nys-tooltip");

// Listen for the 'click' event
tooltip.addEventListener("click", () => {
	console.log("Tooltip clicked");
});

// Listen for the 'focus' event
tooltip.addEventListener("focus", () => {
	console.log("Tooltip is focused");
});

// Listen for the 'blur' event
tooltip.addEventListener("blur", () => {
	console.log("Tooltip lost focus");
});
```
