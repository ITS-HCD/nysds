# Label **Component**

<nys-alert
  type="warning"
  heading="This component should not be used alone. It is already built into form fields.">
</nys-alert>

The **`nys-label`** is a reusable web component for use in New York State digital products. It allows users to add a label to a form field. It can display description and optional/required flags.
