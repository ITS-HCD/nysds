## Options

### Inverted
The `inverted` attribute should be used when the divider is placed on a dark background. It changes the color of the divider to ensure visibility.
<Canvas of={NysDividerStories.Inverted} />

## Usage

### When to use this component
- To separate content displayed vertically
- Set the  `inverted` property when using the component on a dark background

### When to consider something else
- When content is horizontally formatted
