## Events

The `nys-divider` component emits the following events:

1. `nys-click`: Emitted when the divider is clicked.
2. `nys-focus`: Emitted when the divider receives focus.
3. `nys-blur`: Emitted when the divider loses focus.

You can listen to these events using JavaScript:

```js
// Select the divider component
const divider = document.querySelector("nys-divider");

// Listen for the 'nys-click' event
divider.addEventListener("nys-click", () => {
	console.log("Divider clicked");
});

// Listen for the 'nys-focus' event
divider.addEventListener("nys-focus", () => {
	console.log("Divider is focused");
});

// Listen for the 'nys-blur' event
divider.addEventListener("nys-blur", () => {
	console.log("Divider lost focus");
});
```
