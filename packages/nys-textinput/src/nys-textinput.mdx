# Text Input
The `nys-textinput` is a reusable web component for use in New York State digital products. It allows users to input data to be collected.

### Basic Text Input

<Canvas of={NysTextinputStories.Blank} />

### Size

If no size is provided, the `nys-textinput` will fill the width of its parent. Supported sizes are `xs`, `sm`, `md`, `lg`, and `xl`\
If any other input is assigned to `size`, it will be ignored and fit to the parent width.

<Canvas of={NysTextinputStories.Size} />

### Different Input Types
Note: Accepted types are: `text` `email` `number` `password` `search` `tel` `url`\
