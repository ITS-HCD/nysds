# Select
The `nys-select` is a reusable web component for use in New York State digital products. It allows users to select data from a dropdown to be collected.

### Multiple Select

Following the guidance of USWDS, we recommend against using `<select multiple/>` as an input for users to select multiple items from a dropdown. Instead use a checkbox group.
