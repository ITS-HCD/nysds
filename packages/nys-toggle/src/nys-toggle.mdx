# Toggle
The **`nys-toggle`** component is a reusable web component for use in New York State digital products. It allows users to toggle an option on or off.

If an id is not passed, a unique id will be generated.

## Accessibility

The `nys-toggle` component includes the following accessibility-focused features:

- **ARIA Hidden by Default:** If no label is provided, the icon is hidden from screen readers by setting `aria-hidden="true"`.
- **Customizable ARIA Label:** If a label is provided, the component automatically adds an `aria-label` attribute, making the icon accessible to screen readers.
