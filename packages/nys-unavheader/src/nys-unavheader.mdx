# Universal Header

The **`nys-unavheader`** component is a reusable web component for use in New York State digital products. It helps provide users with access to essential links across all pages.

## Usage 

### When to use this component
  - Use `nys-unavheader` to convey that this is a legitimate ny.gov site.

### When to consider something else
  - Do not place the nys-unavheader anywhere other than the top of the page as the topmost header.

The `nys-unavheader` component includes the following accessibility-focused features:

- Proper use of `<header>` and other elements ensures compatibility with assistive technologies.
- Keyboard navigation: Users can tab through all interactive elements of the header.
