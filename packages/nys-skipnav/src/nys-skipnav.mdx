## Events

The `nys-skipnav` component emits the following events:

1. `click`: Emitted when the skipnav is clicked.
2. `focus`: Emitted when the skipnav receives focus.
3. `blur`: Emitted when the skipnav loses focus.

You can listen to these events using JavaScript:

```js
// Select the skipnav component
const skipnav = document.querySelector("nys-skipnav");

// Listen for the 'click' event
skipnav.addEventListener("click", () => {
	console.log("Skipnav clicked");
});

// Listen for the 'focus' event
skipnav.addEventListener("focus", () => {
	console.log("Skipnav is focused");
});

// Listen for the 'blur' event
skipnav.addEventListener("blur", () => {
	console.log("Skipnav lost focus");
});
```
