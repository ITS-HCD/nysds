# Error Message **Component**
The **`nys-errormessage`** is a reusable web component for use in New York State digital products. It allows users to add an error message to a form field. It can display description and optional/required flags.
errormessage
Note: `nys-errormessage` is already built into form components. You do not need to add it separately.

## Options

## Use cases    
Use in form components to display an error to a user if the input is invalid and needs to be modified.
In inputs that are bordered (ie. text inputs and select) do not show the divider. Otherwise add it with `showDivider`
