# Icon

The **`nys-icon`** component is a reusable web component for use in New York State digital products. It provides users with the ability to choose a specific svg icon to use.

### SVG props implementations:
        * Width and Height (default: relative to font size)
            * Custom `nys-icon--size` variants
        * Color (Hex value - default: #000000)
        * Rotation (Degree)
        * Scale

## Icon Property Examples

### Width and Height
> Both width and height they are set **relative** to the font size. 
> To customize the icon size, use the component’s size variants, nys-icon--size-[size number], which range from 3 (1.5rem or 24px) to 9 (4.5rem or 72px).
>
> **IMPORTANT ⓘ** these variant will <ins>override</ins> the already set width and height within this component. 

> Example: change the empty string in *className* to `nys-icon--size-5`

## Accessibility

The `nys-icon` component includes the following accessibility-focused features:

- **ARIA Hidden by Default:** If no label is provided, the icon is hidden from screen readers by setting `aria-hidden="true"`.
- **Customizable ARIA Label:** If a label is provided, the component automatically adds an `aria-label` attribute, making the icon accessible to screen readers.
