# Icon

The **`nys-icon`** component is a reusable web component for use in New York State digital products. It provides users with the ability to choose a specific svg icon to use.

### SVG props implementations:
        * Custom size naming variants
        * Color 
        * Rotation (Degree)

### Size Inheritance
The icon size will inherit the relative font size from its nearest parent. 

<Canvas of={NysIconStories.InheritSize} />

### Size Naming Variant
To scale an icon relative to the inherited font size, pass a size variant like `size=[variant name]` as a prop:
- xs (extra small): 50% of parent font size
- sm (small): 100% of parent font size
- md (medium): 150% of parent font size
- lg (large): 200% of parent font size
- xl (extra large): 250% of parent font size

Each size variant (xs, sm, md, lg, xl) scales the iconâ€™s height relative to the parent font size, from smallest (xs) to largest (xl).

## Accessibility

The `nys-icon` component includes the following accessibility-focused features:

- **ARIA Hidden by Default:** If no label is provided, the icon is hidden from screen readers by setting `aria-hidden="true"`.
- **Customizable ARIA Label:** If a label is provided, the component automatically adds an `aria-label` attribute, making the icon accessible to screen readers.
