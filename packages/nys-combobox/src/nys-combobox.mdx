## Events

The `nys-combobox` component emits the following events:

1. `nys-click`: Emitted when the combobox is clicked.
2. `nys-focus`: Emitted when the combobox receives focus.
3. `nys-blur`: Emitted when the combobox loses focus.

**Event details** \
The eventâ€™s `detail` object includes:
- id (string): The id of the combobox.
- checked (boolean): Whether the combobox is currently checked.

You can listen to these events using JavaScript:

```js
// Select the combobox component
const combobox = document.querySelector("nys-combobox");

// Listen for the 'nys-click' event
combobox.addEventListener("nys-click", (event) => {
  const {id, checked} = event.detail;
	console.log("Combobox (${id}) is clicked. Checked: ${checked}");
});

// Listen for the 'nys-focus' event
combobox.addEventListener("nys-focus", () => {
	console.log("Combobox is focused");
});

// Listen for the 'nys-blur' event
combobox.addEventListener("nys-blur", () => {
	console.log("Combobox lost focus");
});
```
