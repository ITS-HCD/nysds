## Events

The `nys-accordion` component emits the following events:

1. `nys-click`: Emitted when the accordion is clicked.
2. `nys-focus`: Emitted when the accordion receives focus.
3. `nys-blur`: Emitted when the accordion loses focus.

You can listen to these events using JavaScript:

```js
// Select the accordion component
const accordion = document.querySelector("nys-accordion");

// Listen for the 'nys-click' event
accordion.addEventListener("nys-click", () => {
	console.log("Accordion clicked");
});

// Listen for the 'nys-focus' event
accordion.addEventListener("nys-focus", () => {
	console.log("Accordion is focused");
});

// Listen for the 'nys-blur' event
accordion.addEventListener("nys-blur", () => {
	console.log("Accordion lost focus");
});
```
